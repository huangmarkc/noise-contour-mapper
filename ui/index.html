How to use

1. Load the client floor plan (or a PDF figure — page 1 is used).

2. Optionally calibrate the scale using a known dimension, then set an origin point (a building corner works well). Positions are then handled in real units (x right, y up) instead of pixels.

3. Switch to Add points and click each sound-level-meter location; enter the measured dBA. Or import a CSV.

4. The heat map and contours update live. Tune method, bands, and opacity.

Alternative method: place known equipment as noise sources (level at a reference distance) and choose Source model + residuals. Levels are predicted with the inverse square law and energy addition, then calibrated to your measurements by interpolating the residuals. The result matches your readings at each SLM point while following the physics between them. It is a planning/visualization aid, not a substitute for dosimetry.

5. Drag markers (in Pan mode) to fine-tune positions. Edit values in the point list.

6. Export the finished map as PNG for reports, or save the project as JSON to continue later.

Interpolated values are estimates between measurements — verify critical boundaries (e.g., the 85 dBA action level) with additional measurements.
